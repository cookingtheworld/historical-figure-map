# 历史人物地理志

面向 1760-2025 年中国历史人物的个人知识管理工具。通过表单 + 地图点选的方式添加人物生平轨迹，以学术历史风格可视化呈现。

## 功能

- **人物管理**：添加、编辑、删除历史人物基本信息
- **地图轨迹**：在 Leaflet 地图上点选或搜索历史地点，构建人物生平轨迹
- **事件记录**：为每个地点添加多个事件（科举、任职、贬谪等）
- **时间线**：按年份排序查看所有事件
- **著作索引**：自动提取事件中关联的著作，形成索引
- **数据持久化**：浏览器 LocalStorage 自动保存
- **导入/导出**：下载/上传 JSON 数据文件

## 使用方式

双击 `index.html` 在浏览器中打开即可使用，无需任何服务器环境。

## 快速开始

1. 点击「+ 添加」按钮
2. 填写人物基本信息（姓名、朝代、生卒年、籍贯）
3. 点击「保存」→ 进入编辑模式
4. 在编辑面板中点击「+ 添加地点」
5. 搜索县名或直接在地图上点击
6. 确定地点后，添加事件（年份、类型、标题等）
7. 重复添加更多地点，构建完整轨迹
8. 在右侧面板查看时间线和著作索引

## 技术栈

- **地图**：Leaflet.js + OpenStreetMap
- **存储**：浏览器 LocalStorage
- **纯前端**：零后端依赖

## 数据文件

- `data/figures.json`：人物数据（可导出/导入）
- `data/locations.json`：200 个预置历史地点坐标（1760-2025 年高频县）
